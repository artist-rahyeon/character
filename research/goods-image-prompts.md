## 공통 스타일 (처음에 한 번만 설명)

Gemini에 처음 보낼 때 이 스타일 지정을 포함하세요:

```
앞으로 4장의 상품 목업 이미지를 만들 거야.
스타일은 전부 통일해줘:
- 깔끔한 제품 사진 느낌의 일러스트
- 배경은 크림 화이트 단색
- 그림자 최소한으로
- 프리미엄 아동 브랜드 카탈로그 느낌
- 전체적으로 따뜻하고 부드러운 톤
- 텍스트 없이 상품만
```

## 이미지 1: 감정 인형 세트

```
아기 부엉이 캐릭터 봉제 인형 5개가 나란히 놓여있는 제품 사진 스타일 일러스트.

5개의 인형은 같은 캐릭터인데 각각 다른 표정:
1. 활짝 웃는 표정 (기쁨)
2. 눈이 축 처지고 눈물 한 방울 (슬픔)
3. 볼이 부풀고 눈썹이 올라간 표정 (화남)
4. 눈이 동그랗게 커진 표정 (놀람)
5. 눈이 반쯤 감기고 편안한 미소 (평온)

인형은 부드러운 브라운 색상의 봉제 질감.
크림 화이트 배경. 프리미엄 아동 브랜드 카탈로그 느낌.
인형 크기는 손바닥 정도로 작고 귀여운 사이즈.
```

## 이미지 2: 프리미엄 워크북

```
프리미엄 아동 교육 워크북 3권이 살짝 겹쳐서 놓여있는 제품 사진 스타일 일러스트.

워크북 표지 디자인:
- 매트한 질감의 표지
- 크림, 베이지, 연한 브라운 톤의 미니멀한 표지
- 각 표지에 귀여운 아기 부엉이 캐릭터가 작게 그려져 있음
- 하나는 연한 코랄, 하나는 연한 블루, 하나는 연한 그린 포인트

옆에 색연필 몇 자루가 자연스럽게 놓여있음.
크림 화이트 배경. 프리미엄 아동 브랜드 카탈로그 느낌.
```

## 이미지 3: 마음숲 플레이 세트

```
나무 소재의 아동 장난감 세트가 놓여있는 제품 사진 스타일 일러스트.

세트 구성:
- 작은 나무 피규어 나무 5그루 (각각 다른 색: 코랄, 블루, 그린, 머스타드, 라벤더)
- 아기 동물 미니어처 5개 (부엉이, 햄스터, 여우, 토끼, 곰) — 나무 소재, 심플한 형태
- 동그란 초록색 펠트 매트 (숲 바닥 역할)

전체적으로 북유럽 원목 장난감 느낌.
크림 화이트 배경. 프리미엄 아동 브랜드 카탈로그 느낌.
```

## 이미지 4: 생활용품 세트

```
아이용 생활용품 3~4개가 자연스럽게 배치된 제품 사진 스타일 일러스트.

구성:
- 아기 부엉이 캐릭터가 그려진 스테인리스 물컵 (뚜껑 달린)
- 같은 캐릭터의 수저 세트 (케이스 포함)
- 작은 미니 가방 또는 파우치 (부엉이 캐릭터 자수 또는 프린트)

색상은 크림, 베이지, 연한 브라운 톤.
캐릭터는 작고 심플하게.
크림 화이트 배경. 프리미엄 아동 브랜드 카탈로그 느낌.
```
