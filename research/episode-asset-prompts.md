## 이미지 4장

### 이미지 1: 햄햄이 화난 표정

```
아까 만든 햄스터 캐릭터를 같은 스타일로 다시 그려줘.

이번에는 화가 잔뜩 난 표정이야:
- 양쪽 볼이 빵빵하게 부풀어올라 있어 (평소보다 훨씬 크게)
- 눈썹이 가운데로 모여서 찡그린 표정
- 입을 꾹 다물고, 살짝 삐죽
- 볼과 귀 끝이 살짝 붉어진 느낌
- 작은 주먹을 불끈 쥐고 있는 자세
- 발은 동동 구르는 듯 한쪽이 살짝 들린 포즈

배경은 깨끗한 크림 화이트 단색.
기존 햄스터와 완전히 같은 캐릭터인데 표정만 다르게.
텍스트 없이 캐릭터만.
```

### 이미지 2: 햄햄이 진정된 표정

```
같은 햄스터 캐릭터, 같은 스타일.

이번에는 화가 풀리고 마음이 편안해진 표정:
- 볼이 원래 크기로 돌아와 있어 (부풀지 않은 상태)
- 눈을 부드럽게 반쯤 감고, 눈꼬리가 살짝 내려간
- 입이 작게 미소 짓고 있어
- 양손을 배 위에 가만히 올려놓은 편안한 자세
- 전체적으로 힘이 빠지고 이완된 느낌
- 볼에 은은한 복숭아빛 볼터치

배경은 깨끗한 크림 화이트 단색.
텍스트 없이 캐릭터만.
```

### 이미지 3: 포포 숨 쉬는 모습

```
아까 만든 곰 캐릭터, 같은 스타일.

이번에는 눈을 감고 깊은 숨을 내쉬고 있는 모습:
- 눈을 완전히 감고 있어. 속눈썹이 살짝 보임.
- 입이 작게 동그랗게 열려서 "후~" 하고 숨을 내쉬는 중
- 배가 살짝 부풀어 있는 느낌 (숨을 가득 담고 있는)
- 양팔은 자연스럽게 아래로 내려놓은 편안한 자세
- 주변에 아주 작은 빛 알갱이 3~4개가 떠다니는 느낌 (숨결 표현)
- 표정이 매우 평화롭고 편안

배경은 깨끗한 크림 화이트 단색.
텍스트 없이 캐릭터만.
```

### 이미지 4: 마음숲 나무 (바람개비 열매)

```
크림 화이트 배경에 한 그루의 작은 나무를 그려줘.

- 나무 줄기는 따뜻한 브라운, 살짝 구불구불한 형태
- 나뭇잎은 둥글둥글한 형태로, 연두색에서 노란색으로 그라데이션
- 가지 3개에 각각 작은 바람개비 모양 열매가 하나씩 달려 있어
- 바람개비는 파스텔 색상 (코랄, 하늘색, 연보라)
- 나무 아래에 작은 풀과 꽃이 2~3송이
- 나무 크기는 화면의 60% 정도 차지

전체적으로 아동 그림책 일러스트 느낌.
따뜻하고 부드럽고 동화적인 분위기.
텍스트 없이.
같은 일러스트 스타일 유지.
```

## 영상 3개

### 영상 1: 블록이 와르르 무너지는 장면 (5초)

```
아동 그림책 스타일의 부드러운 애니메이션 영상을 만들어줘. 5초 분량.

장면:
크림 화이트 배경에 알록달록한 나무 블록 탑이 쌓여 있어.
블록은 파스텔 색상 (코랄, 하늘색, 연두, 노랑, 라벤더) 으로 7~8개 쌓여있어.
탑이 흔들리다가 와르르 무너져. 블록들이 사방으로 흩어짐.
무너질 때 화면이 아주 살짝 흔들리는 느낌.

분위기:
- 무섭거나 폭력적이지 않게. 귀엽고 아기자기하게 무너지는 느낌.
- 아동 그림책 일러스트 스타일의 부드러운 애니메이션
- 배경은 크림 화이트
- 텍스트 없이
```

### 영상 2: 곰 캐릭터가 깊은 숨을 쉬는 장면 (10초)

```
아동 그림책 스타일의 부드러운 애니메이션 영상을 만들어줘. 10초 분량.

장면:
아까 만든 아기 곰 캐릭터가 화면 가운데에 있어.
곰이 눈을 감고 천천히 깊은 숨을 쉬고 있어.

호흡 사이클 (2회 반복):
1. 숨을 들이쉼 — 배가 천천히 부풀어오름 (3초)
2. 잠깐 멈춤 (1초)
3. 숨을 내쉼 — 배가 천천히 줄어듦 (3초)

곰 주변에 작고 밝은 빛 알갱이들이 숨결과 함께 천천히 떠다님.
들이쉴 때 빛 알갱이들이 곰 쪽으로 모여들고,
내쉴 때 바깥으로 퍼져나감.

분위기:
- 매우 평화롭고 잔잔함
- 아동 그림책 일러스트 스타일
- 배경은 아주 연한 민트/크림 그라데이션
- 텍스트 없이
```

### 영상 3: 마음숲에 나무가 자라나는 장면 (8초)

```
아동 그림책 스타일의 부드러운 애니메이션 영상을 만들어줘. 8초 분량.

장면:
연한 초록빛 풀밭 위에 작은 새싹이 하나 돋아남.

성장 과정:
1. 땅에서 작은 새싹이 "쏙" 올라옴 (2초)
2. 새싹이 줄기로 자라고, 가지가 좌우로 뻗어남 (3초)
3. 가지에 둥근 나뭇잎이 하나둘 피어남 (2초)
4. 마지막에 가지 끝에 작은 바람개비 모양 열매가 달리고, 살짝 빙글 돌아감 (1초)

바람개비 열매는 파스텔 색상 (코랄, 하늘색, 연보라).
나무가 다 자라면 작은 새 한 마리가 날아와서 가지에 앉음.
전체적으로 반짝이는 작은 빛 입자가 나무 주변에 떠다님.

분위기:
- 따뜻하고 희망적이고 동화적
- 아동 그림책 일러스트 스타일
- 텍스트 없이
```
