# 공부엉즈 캐릭터 — AI 이미지 생성 프롬프트

> ChatGPT (DALL-E 3) 용으로 최적화
> 모든 프롬프트에 동일한 스타일 지정을 넣어서 5캐릭터의 일관성을 확보

---

## 사용법

1. ChatGPT에 아래 프롬프트를 하나씩 복붙
2. 결과가 마음에 안 들면 "좀 더 둥글게", "눈을 더 크게", "색을 더 따뜻하게" 등으로 조정
3. 마음에 드는 결과가 나오면 저장
4. 5캐릭터 전부 만든 후 톤이 안 맞는 것만 재생성

## 추가 팁

### 톤 조정 시 ChatGPT에 말할 수 있는 것들
- "Make it rounder and softer" — 더 둥글게
- "Make the eyes 30% bigger" — 눈 더 크게
- "Warmer colors, less saturated" — 더 따뜻하고 부드러운 색
- "More like a Scandinavian children's book illustration" — 북유럽 그림책 스타일로
- "Keep exactly the same style but change the character to..." — 스타일 유지하면서 캐릭터만 변경

### 일관성 유지 팁
- ChatGPT 한 대화 안에서 5캐릭터를 연속으로 만들면 스타일이 비슷하게 유지됨
- 첫 번째(공부엉)가 마음에 들면 "Keep this exact same art style" 을 다음 프롬프트에 추가
- 결과물 중 스타일이 튀는 게 있으면 마음에 드는 이미지를 첨부하고 "Match this style exactly"로 재생성

### 해상도
- ChatGPT DALL-E 3: 기본 1024x1024. "Generate in the highest resolution possible" 추가
- 나중에 업스케일 필요하면 upscale.media 같은 무료 도구로 2~4배 키울 수 있음
