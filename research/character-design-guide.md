# 공부엉 캐릭터 디자인 가이드라인

## Part 1. 디자인 리서치

### 1-1. 프리미엄 + 유아 매력을 동시에 잡은 레퍼런스

| 캐릭터 | 특징 | 참고 포인트 |
|--------|------|-------------|
| **Miffy (Dick Bruna)** | 극도로 미니멀한 선, 6색 제한, 점 눈 + 십자 입 | 단순함이 프리미엄으로 읽힘. 감정 투영 가능 |
| **Bluey (Ludo Studio)** | 둥근 직사각형 기반 형태, 단순화된 비율, 플랫 카메라 | 단순하지만 생동감 있는 표정과 제스처 |
| **Hey Bear Sensory** | 부드러운 4K 애니메이션, 예측 가능한 움직임, 고대비 색상 | 시각적 안정감 + 감각 친화적 디자인 |
| **Momonsters** | 둥글고 부드러운 3D 형태, 밝은 단색 체색 | 각 캐릭터가 하나의 감정/성격 대표 |
| **Waffles + Mochi (Netflix)** | 친근한 비율, 따뜻한 색감, 교육 + 재미 균형 | 교육 캐릭터가 "완벽한 선생" 아닌 설정 |
| **Puffin Rock** | 수채화풍, 자연 배경, 부드러운 내레이션 | 프리미엄 자연주의 + 유아 친화 톤 |

### 1-2. 2024~2026 유아 캐릭터 디자인 트렌드

1. **자연주의 팔레트**: 베이지, 탄, 올리브그린, 클레이, 크림 — 이솝/킨포크 감성과 직접 연결
2. **유기적 형태 (Organic Forms)**: 자연에서 온 불규칙하지만 부드러운 곡선, 단순화된 실루엣
3. **감각 친화 디자인**: 과도한 자극 배제, 예측 가능한 시각 패턴, 부드러운 전환
4. **포용적 캐릭터**: 다양성을 내재한 디자인, 감정 투영이 가능한 단순 표정
5. **"Grow Gently" 무드**: 차분하고 감각적인, 따뜻한 뉴트럴 기반의 미니멀리즘

### 1-3. 성공 사례 디자인 분석

#### Bluey
- **형태 언어**: 세계 전체가 "크고 친근한 둥근 직사각형"으로 통일
- **비율**: 미니어처화된 비율 → 귀여움과 몰입 유도
- **배경**: 극도로 단순화, 그래픽적 표현 (사실적 묘사 배제)
- **카메라**: 정면, 평면적 연출이 기본

#### Miffy (Dick Bruna)
- **원칙**: 아이의 시각 세계는 혼란스럽지 않아야 한다
- **제한**: 최대 6색, 굵은 검은 외곽선, 최소한의 디테일
- **심리**: 단순한 표정 → 아이가 자기 감정을 투영
- **영향**: 드 스틸, 마티스, 레제의 명확한 형태와 대담한 색

#### Hey Bear Sensory
- **설계**: 부드럽고 예측 가능하고 위협적이지 않은 애니메이션
- **전환**: 에너지 있는 시작 → 차분한 마무리 구조
- **접근성**: 감각 민감 아동도 편안하게 시청 가능

### 1-4. 부엉이 캐릭터 디자인 시 참고 사항

- 부엉이의 **큰 눈**은 유아 캐릭터와 자연스럽게 어울림 (호기심, 경이로움 표현에 유리)
- 둥근 얼굴 + 짧은 몸통이 기본 → 3~5세 아이에게 안정감
- 따뜻한 브라운/크림 체색은 자연주의 트렌드와 정확히 부합
- 날개는 손처럼 활용 가능 → 제스처/감정 표현의 도구
- 뿔깃(귀처럼 보이는 부분)은 감정 표현 보조 장치로 활용 가능

## Part 2. 공부엉 디자인 가이드라인

### 2-1. 얼굴 및 전신 비율

| 항목 | 가이드 |
|------|--------|
| 머리:몸 비율 | **1:0.8** (머리가 몸보다 약간 큼 — 유아 친화적 두신 비율) |
| 눈 크기 | 얼굴 면적의 약 35~40% 차지. 동그란 원형, 홍채 크게 |
| 눈 간격 | 눈 하나 폭 정도의 간격 |
| 부리 | 작고 짧은 삼각형. 얼굴 중앙 하단. 색상: 머스타드 옐로우 |
| 뿔깃 | 머리 위 양쪽에 작은 삼각형 2개. 감정에 따라 각도 변화 |
| 전체 높이 | 약 2~2.5등신 (머리:몸 비율 1:0.8 기준, 머리가 몸보다 약간 큰 유아 친화 비율) |

### 2-2. 색상 팔레트

#### 메인 체색
| 용도 | 색상 | HEX |
|------|------|-----|
| 몸통 메인 | 웜 브라운 | `#8B6F4E` |
| 배 부분 | 크림/아이보리 | `#F5EDE3` |
| 날개 그라데이션 | 다크 브라운 | `#6B5340` |

#### 포인트 컬러
| 용도 | 색상 | HEX |
|------|------|-----|
| 부리, 발 | 머스타드 | `#D4A84B` |
| 볼터치 | 피치 핑크 | `#F4B8A5` |
| 눈 하이라이트 | 화이트 | `#FFFFFF` |

#### 배경/환경 팔레트
| 용도 | 색상 | HEX |
|------|------|-----|
| 기본 배경 | 웜 화이트 | `#FAFAF8` |
| 보조 배경 | 세이지 그린 | `#C5D1C0` |
| 액센트 | 소프트 테라코타 | `#D4886B` |

### 2-3. 선 스타일

- **외곽선 없음** — 면(shape)으로만 구성
- 형태 구분은 색상 차이와 그림자로 표현
- 필요 시 매우 부드러운 그림자(soft shadow)로 깊이감 부여
- Miffy의 굵은 외곽선과 명확히 차별화
- 뽀로로의 검은 외곽선 스타일 완전 배제

### 2-4. 표정 표현 방식

| 요소 | 기본 | 변형 |
|------|------|------|
| **눈** | 크고 동그란 원, 위 1/3에 하이라이트 점 | 반달눈(웃을 때), 별눈(놀랄 때), 물결눈(졸릴 때) |
| **입** | 작은 미소 곡선 (부리 아래) | 동그란 O(놀람), 삐죽(삐침), 넓은 미소(기쁨) |
| **볼** | 항상 은은한 피치 핑크 볼터치 | 감정 강도에 따라 채도 변화 |
| **뿔깃** | 살짝 바깥으로 벌어짐 (기본) | 쫑긋(호기심), 눕힘(슬픔), 팔랑(신남) |
| **날개** | 몸 옆에 자연스럽게 | 들어올림(인사), 감싸기(위로), 펄럭(흥분) |

### 2-5. 체형

- **전체 실루엣**: 달걀형(위가 큰 타원)
- **머리**: 거의 원형, 약간 위아래 납작
- **몸통**: 짧고 둥근 배 (아기 체형)
- **날개**: 몸통 양쪽, 짧고 둥근 끝. 팔처럼 활용
- **발**: 짧고 작은 발 2개, 머스타드 색
- **꼬리**: 없거나 매우 짧은 뭉툭한 형태

### 2-6. 기본 포즈

1. **기본 정면**: 양 날개 자연스럽게 내리고, 살짝 고개 기울임, 미소
2. **인사**: 한쪽 날개 들어올림, 눈 반달
3. **호기심**: 몸을 약간 앞으로, 뿔깃 쫑긋, 눈 크게
4. **실수/당황**: 눈 동그랗게, 날개로 입 가리기, 뿔깃 눕힘
5. **격려**: 양 날개 앞으로 내밀기, 넓은 미소

### 2-7. 피해야 할 것

| 피해야 할 요소 | 이유 |
|----------------|------|
| 검은 두꺼운 외곽선 | 뽀로로/핑크퐁과 유사해짐 |
| 과도한 디테일 (깃털 하나하나) | 미니멀 브랜드 톤과 불일치 |
| 날카로운 부리/발톱 | 유아에게 위협적 |
| 보라/검정 체색 | "밤/무서움" 연상. 부엉이 = 무서운 이미지 강화 |
| 안경/졸업모 | "가르치는" 이미지 → "같이 배우는 친구" 설정과 충돌 |
| 너무 사실적인 부엉이 형태 | uncanny. 유아 거부감 |
| 기존 부엉이 캐릭터와 유사한 포즈 | Owl (Winnie the Pooh), Owlette (PJ Masks) 등과 차별화 필요 |
| 차가운 색조 (파랑, 회색 위주) | 브랜드 톤 "따뜻함"과 불일치 |

## Part 3. 디자인 원칙 요약

### 공부엉 디자인 5원칙

1. **Warm Simplicity** — 따뜻한 단순함. 선 없이 면으로, 6색 이내로.
2. **Curious Eyes** — 눈이 캐릭터의 핵심. 크고, 빛나고, 감정이 담긴.
3. **Soft & Round** — 모든 형태는 둥글게. 각진 부분 없이.
4. **Growing Together** — 완벽하지 않은 체형. 약간의 비대칭이 주는 친근함.
5. **Nature-Born** — 색상, 형태, 텍스처 모두 자연에서 온 것처럼.
