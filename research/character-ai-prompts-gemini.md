# 공부엉즈 캐릭터 — Gemini(나노바나나) 이미지 생성 프롬프트

> Gemini 이미지 생성용으로 최적화
> 캐릭터 일관성 유지를 위해 한 대화 안에서 순서대로 생성할 것

---

## 사용법

1. Gemini 열기 (gemini.google.com)
2. **Step 1**의 스타일 지정 프롬프트를 먼저 보내기
3. 공부엉 이미지가 마음에 들면, 같은 대화에서 나머지 캐릭터를 순서대로 생성
4. 마음에 안 드는 부분은 "눈을 더 크게", "색을 더 따뜻하게" 등 한국어로 조정 가능
5. 마음에 드는 캐릭터가 나오면 "이 캐릭터로 웃는 표정, 슬픈 표정, 놀란 표정 3장 만들어줘"로 표정 시트 제작

---

## Step 1: 공부엉 (메인 캐릭터) + 스타일 확립

이 프롬프트가 제일 중요합니다. 여기서 스타일이 잡히면 나머지도 따라갑니다.

```
프리미엄 유아 교육 앱을 위한 캐릭터 일러스트를 그려줘.

스타일:
- 고급 아동 그림책 일러스트 느낌
- 외곽선 없이 부드러운 면(fill)으로만 구성
- 따뜻한 파스텔 색감
- 배경은 깨끗한 크림 화이트 단색
- 캐릭터만 정중앙에 정면으로
- 머리가 크고 몸이 작은 2:2.5 등신 비율
- 매우 큰 눈에 하이라이트가 반짝이는
- 볼에 은은한 복숭아색 볼터치
- 텍스트 없이 캐릭터만

캐릭터: 아기 부엉이
- 따뜻한 브라운 몸통, 크림색 배
- 머리 위에 작은 뿔깃 2개 (비대칭 — 왼쪽이 살짝 작고 오른쪽이 큼)
- 커다랗고 동그란 눈. 갈색 홍채, 검은 동공, 흰 하이라이트 2개 (큰 거 + 작은 거)
- 작은 머스타드 노란색 부리, 살짝 아래로 구부러진
- 양쪽에 작은 날개 (몸보다 약간 어두운 브라운)
- 작은 머스타드 노란색 발
- 크림색 배에 V자 깃털 무늬 2~3개
- 표정: 호기심 가득, 다정한 미소, 고개를 살짝 갸웃
- 전체적으로 둥글고 부드럽고 포근한 느낌
```

→ 결과가 마음에 들면 다음으로. 안 들면 조정 예시:
- "좀 더 둥글고 통통하게 해줘"
- "눈을 30% 더 크게"
- "색을 좀 더 따뜻하게, 채도를 살짝 올려줘"
- "좀 더 아기 같은 느낌으로"

## Step 2: 햄햄이 (햄스터, 감정 담당)

```
방금 그린 부엉이 캐릭터와 완전히 같은 스타일로 새 캐릭터를 그려줘. 스타일, 색감 톤, 배경, 등신 비율, 눈 표현 방식 모두 동일하게 유지해줘.

캐릭터: 아기 햄스터
- 부드러운 피치/살구색 둥근 몸통, 흰색 배
- 양쪽 볼이 빵빵하게 부풀어 있음 (햄스터 볼주머니). 볼에 분홍 볼터치.
- 작고 둥근 귀 2개, 귀 안쪽은 연한 핑크
- 커다랗고 반짝이는 눈 (부엉이와 같은 스타일)
- 작은 분홍색 코
- W자 모양으로 올라간 활짝 웃는 입
- 양쪽에 가느다란 수염 3개씩
- 앞에 모으고 있는 작은 분홍색 손
- 표정: 활짝 웃는, 감정이 풍부한, 행복 에너지 가득
- 이 캐릭터는 감정을 담당하는 친구야
```

---

## Step 3: 부비 (여우, 사고력 담당)

```
같은 스타일을 유지하면서 새 캐릭터를 그려줘.

캐릭터: 아기 여우
- 따뜻한 오렌지/앰버색 몸통, 크림색 배
- 크고 뾰족한 삼각형 귀, 귀 안쪽은 연한 살구색
- 양쪽 볼에 크림색 패치 (여우 특유의 밝은 뺨)
- 작은 검은 삼각형 코
- 눈이 살짝 가로로 긴 타원형. 한쪽 눈썹이 올라가 있는 호기심 표정.
- 살짝 씩 웃는 입
- 오른쪽에 크고 풍성한 꼬리. 오렌지에서 끝부분이 흰색으로.
- 표정: 호기심 넘치는, 장난기 있는, "왜?"라고 물어보는 얼굴
- 이 캐릭터는 사고력과 탐구를 담당하는 친구야
```

---

## Step 4: 또리 (토끼, 사회성 담당)

```
같은 스타일을 유지하면서 새 캐릭터를 그려줘.

캐릭터: 아기 토끼
- 부드러운 라벤더/블루그레이 몸통, 흰색에 가까운 밝은 배
- 길고 큰 귀 2개. 왼쪽 귀는 쫑긋 서 있고, 오른쪽 귀는 끝이 살짝 접혀 있음. 귀 안쪽은 핑크.
- 작은 핑크색 하트 모양 코
- 눈꼬리가 살짝 내려간 다정한 눈 (부드럽고 따뜻한 인상)
- 작고 다정한 미소
- 은은한 핑크 볼터치
- 작은 앞발을 가지런히 모으고 있는 자세
- 뒤에 살짝 보이는 둥근 흰색 솜방울 꼬리
- 표정: 다정한, 경청하는, 조용히 웃는
- 이 캐릭터는 사회성과 배려를 담당하는 친구야
```

---

## Step 5: 포포 (곰, 안정/수면 담당)

```
같은 스타일을 유지하면서 마지막 캐릭터를 그려줘.

캐릭터: 아기 곰
- 5캐릭터 중 가장 통통하고 둥근 체형
- 부드러운 카키/올리브 그린 몸통, 밝은 카키색 배
- 작고 둥근 귀 2개, 귀 안쪽은 연한 색
- 둥근 갈색 타원형 코, 작은 하이라이트
- 눈이 반쯤 감겨 있음 — 눈꺼풀이 반쯤 내려와서 졸린 표정. 이게 이 캐릭터의 핵심 특징.
- 느긋하고 편안한 미소. 입이 살짝 열려 있어도 좋음.
- 따뜻한 핑크 볼터치
- 짧고 통통한 팔다리
- 표정: 졸린, 느긋한, 평화로운, 포근한
- 이 캐릭터는 휴식과 안정을 담당하는 친구야
```

## Step 6: 5캐릭터 단체샷

5캐릭터 전부 마음에 들면 이걸 보내세요:

```
방금 만든 5캐릭터(부엉이, 햄스터, 여우, 토끼, 곰)를 한 화면에 나란히 배치해서 그려줘. 같은 스타일 유지. 가운데에 부엉이(메인), 양쪽에 나머지 4캐릭터. 배경은 깨끗한 크림 화이트. 모두 정면을 보고 있고, 각자의 대표 표정을 짓고 있어.
```

---

## Step 7: 표정 시트 (공부엉 기준)

메인 캐릭터 표정 변화를 만들고 싶으면:

```
공부엉(부엉이) 캐릭터를 같은 스타일로 6가지 표정을 2x3 그리드로 그려줘:
1. 기본 (호기심 가득 미소)
2. 기쁨 (눈이 초승달 모양으로 웃는)
3. 슬픔 (눈이 축 처지고 눈물 한 방울)
4. 놀람 (눈이 동그랗게 커진)
5. 화남 (볼이 부풀고 눈썹이 올라간)
6. 졸림 (눈이 반쯤 감긴, 포포처럼)
배경은 크림 화이트, 각 표정 아래에 한글로 감정 이름 써줘.
```

## 조정 팁 (한국어로 Gemini에게 말하기)

스타일이 안 맞을 때:
- "외곽선을 완전히 없애줘. 면으로만 표현해"
- "좀 더 미니멀하고 깔끔하게"
- "북유럽 아동 그림책 스타일로"

캐릭터가 안 귀여울 때:
- "머리를 더 크게, 몸을 더 작게"
- "눈을 얼굴의 40%까지 키워줘"
- "전체적으로 더 아기 같은 비율로"
- "더 둥글고 말랑말랑한 느낌으로"

색감이 안 맞을 때:
- "채도를 살짝 올리되, 파스텔 느낌은 유지"
- "색이 너무 차가워. 더 따뜻하게"
- "배경에 아무것도 넣지 마. 크림 화이트 단색만"

일관성이 안 맞을 때:
- "첫 번째 부엉이 캐릭터와 정확히 같은 스타일로 다시 그려줘"
- "눈 표현 방식을 부엉이랑 통일해줘"
